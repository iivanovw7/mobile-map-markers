## mobile-map-markers

---
### Configuration

Google Maps ([API](https://developers.google.com/maps/documentation/javascript/reference)) key is needed to run this application:

Key is stored in application root folder inside `./.env` in the similar way:
```
GOOGLE_MAPS_KEY=...
```
